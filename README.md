MVP версия конфигурации "Салон красоты"
1. Внедрены стандартные подсистемы БСП (редакция 3.1 версия 3.1.11.189)
  - Базовая функциональность
  - Обновление версий ИБ
  - Пользователи
  - Подсистема Организации
  - Подсистема Печать
  - Дополнительные отчеты и обработки
В коде использованы функции БСП
2. Внедрены справочники, документы, регистры и вспомогательные обьекты.
3. Внедрен оперативный учет с контролем остатков.
4. Внедрен бухгалтерский учет.
5. В регистре цены номенклатуры реализована автоматическая загрузка прайс-листа с Excel (Табличный документ должен содержать колонки №, Номенклатура, Код, Цена).
6. Реализованы Web сервисы (REST API) users и customerOrders (подробнее в конфигурации MobileApp).
7. В докменте реализация, печатная форма "Товарная накладная" реализована в виде внешней обработки, подключенной к конфигурации.

Работа Web сервисов протестирована с помощью Postman.
